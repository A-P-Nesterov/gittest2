работа с git

для добавления всех файлов пишем команду git add . для добавления конкретного файла пишем git add "имя файла" для просмотра статуса измегний файлов пишем команду git stus для подготовки файлов к отправке в репозиторий пишем команду git commit -m "название изменений" для отправки файлов репозиторий пишем команду git push
